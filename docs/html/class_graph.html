void Graph::fordFulkerson ( Graph & g,
const std::string & source,
const std::string & sink )

Implement the Ford-Fulkerson algorithm to find the maximum flow in the graph.

This function applies the Ford-Fulkerson algorithm to find the maximum flow from the source vertex to the sink vertex in the given graph. It initializes the flow on all edges to zero, then iteratively finds augmenting paths using BFS and updates the flow along these paths until no augmenting path exists. It returns the maximum flow value found.

Parameters
gReference to the graph on which the algorithm is applied.
sourceThe info attribute of the source vertex.
sinkThe info attribute of the sink vertex.
